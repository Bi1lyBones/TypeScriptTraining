ПРЕИМУЩЕСТВА TypeScript:

1. Строгая типизация - позволяет выявлять соответсвие всех типов при компиляции кода. 
                
2. TypeSafety - свойство TypeScript, при которых ошибки ловятся на ранних этапах рахработки.

3. Autocomplete - поддержка автозаполнения, помогает быстрее писать код, автозаполнение 
                  предлагает только те варианты, которые соответствуют типам. Улучшает
                  понимание кода.

4. Рефакторинг - безопасное и и эффективное изменение кода сминимальным риском возникновения 
                 ошибок.

5. Гибкость и масштабируемость - позыоляет работать со сложными проектами, упрощая понимание
                                 структур данных в проекте.


НЕДОСТАТКИ TypeScript:

1. Скорость кодирования - более сложная компиляция кода. 

2. Обучение - JS узучить с нуля проще, чем TS. Сложное изучение.

3. Отсутсвие контроля над всеми типами данных.

4. Cложность при использовании сторонних библиотек, с некоторыми пакетами TS взаимодействует 
   не оптимально.

5. Дополнительная конфигурация - подготовка проекта может потребовать дополнительных шагов 
                                 для настройки компилятора и файлов.
